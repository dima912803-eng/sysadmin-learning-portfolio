# Interview Stories

## Stable Manager Infrastructure Investigation

Situation:

Given access to a production environment supporting a horse management application.

Task:

Understand architecture and operational components.

Actions:

- Connected through SSH
- Investigated PM2
- Reviewed Nginx
- Analyzed backup strategy
- Investigated CPU spikes

Result:

Created technical documentation and identified areas requiring further investigation.

Lessons Learned:

Production systems should be understood as interconnected components rather than individual services.
